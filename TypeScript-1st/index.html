基础类型

any

类似Object, 区别在于Any可调用任意类型的方法，而Object只能赋值
是所有类型的父类?

void

表示没有任何类型，没有返回值时使用。
只能赋值为undefined 和 null

null & undefined

是所有类型的子类, 默认可赋值给任何类型

如果使用了 --strictNullChecks

never

表示永不存在的值的类型，比如总会抛出异常，或根本不会有返回值，或箭头函数表达式的返回值类型。
变量也可以是never类型，当它被永不为真的类型保护所约束。
是所有类型的子类，也可以赋值给任何类型。除了本身没有任何类型可以赋值给never，即使any也不可以。
